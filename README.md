## ✨ About

This project is a **stunning 3D interactive website** designed and developed for **Techfest — IIT Bombay's annual Science & Technology Festival**, recognized as *Asia's largest*. It was built in collaboration with **Dr-D** as an exclusive digital showcase for **Techfest X**.

The experience fuses cutting-edge WebGL rendering, scroll-driven animations, and glassmorphic UI to deliver a festival landing page that feels truly out of this world.

> 🎉 **Built for: IIT Bombay Techfest X × Dr-D**

---

## 🎯 Features

- **🌐 3D WebGL Canvas** — Fully interactive Three.js scene with animated particle systems, floating geometries, and real-time mouse tracking
- **🎬 Scroll-Driven Animations** — Cinematic scroll sequences powered by GSAP ScrollTrigger
- **🔮 Glassmorphism UI** — Modern frosted-glass cards, panels, and overlays
- **🤖 Robowars Section** — Dedicated arena for India's premier robot combat competition
- **🧠 AI & Workshops** — Highlights for Techfest's flagship AI, ML, and tech workshops
- **📱 Fully Responsive** — Seamlessly adapts from mobile to ultra-wide displays
- **⚡ High Performance** — Optimized render loop with requestAnimationFrame and object pooling
- **🎨 Custom Cursor** — Magnetic cursor with glowing trail effect
- **🌌 Dynamic Starfield** — Procedurally generated star backgrounds with parallax depth

## 🚀 Getting Started

No build step required — this is a pure HTML/CSS/JS project.

### Run Locally

```bash
# Clone the repository
git clone https://github.com/your-username/techfest-3d-iitb.git

# Navigate into the project directory
cd techfest-3d-iitb

# Open in browser (macOS)
open index.html

# Or serve with a simple HTTP server (recommended for best WebGL performance)
npx serve .
```

> **Note:** For optimal 3D performance, serve via a local HTTP server rather than opening `index.html` directly as a file.
